DOM Level 0

Inline

Inline events are what we've done so far. We have an element and place an event handler on it. For example, with the onclick attribute: click me. Now check the box and click again.

Traditional

Instead of "inline" we attach and detach event handlers using JavaScript. This is likely the best approach as it is 100% cross-browser compatible. The onload() function in this page is a good example. We could also do body.onload() or document.onload(). In general window is the best to assure everything is ready to go.
